# Интернет-магазин
## Описание
Интернет-магазин
## Установка
1. Клонирование приложения:
```
git clone https://github.com/olegminntimer/online_store.git
```
2. Установка зависимостей:
```
poetry install
```
## Использование
1. Запуск магазина:
```
python manage.py runserver
```
